(nord, sud, est, ouest).

3. On cherche quel pixel de la texture on veut afficher aux coordonnes (x, y)

4. if (buffer[y][x] == 0x0 && y < screenHeight / 2)
                put_pixel_to_image(glob->image, x, y, GREEN);
            else if (buffer[y][x] == 0x0 && y > screenHeight / 2)
                put_pixel_to_image(glob->image, x, y, WHITE);
